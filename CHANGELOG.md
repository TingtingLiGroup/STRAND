# Changelog

## 0.1.0 (2026-06-27)

Initial public release.

- RNA localization pattern classification (8-class XGBoost with Foci fallback)
- Subcellular compartment detection (SOM-based, GPU optional)
- RNA colocalization analysis (InSTAnT-based PP + CPB)
- Standardized PKL input format
- 4 CLI commands: `subcellfeat`, `subcellfeat-pattern`, `subcellfeat-compartment`, `subcellfeat-coloc`
